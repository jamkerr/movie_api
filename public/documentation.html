YMDB API - Your Movie Database API

With YMDB you can search through movies and save your favourites. This REST API lets you interface with this app as a web service.

Endpoints

Name Descritpion Method URL Parameters Request Format Response Format
All Movies Return all movies. GET /movies None None Returns an array of objects. 
                
[
    {
        "title": "The Adventures of Priscilla, Queen of the Desert",
        "director": "Stephan Elliott"
    },
    {
        "title": "Everything Everywhere All at Once",
        "director": [
            "Daniel Kwan",
            "Daniel Scheinert"
    ]
    }
]
One Movie Return a movie by title. GET /movies/title title None Returns an object. 
                
{
"title": "The Truman Show",
"director": "Peter Weir"
}
Genre Return data about a genre. GET /genre genre None Returns a confirmation string.
Director Return data about a director. GET /director director None Returns a confirmation string.
Register Allow a new user to register. POST /users None Requires a JSON object with the key "username". 
                
{
"username":"Gregor"
}
Returns an object. 
                
{
"username": "Gregor",
"favorites": []
}
Update Username Allow a user to update their username. PUT /users/username username Reqires a JSON object with the key "newname". 
                
{"newname":"Hortence"}
Returns a confirmation string.
Add Favorite Movie Allow a user to add a movie to their favorites. PUT /users/username/favorites username Requires a JSON object with the key "title". 
                
{
"title":"filmTitle",
"director":"directorName"
}
Returns a confirmation string.
Remove Favorite Movie Allow a user to remove a movie from their favorites. DELETE /users/username/favorites username Requires a JSON object with the key "title". 
                
{
"title":"filmTitle",
"director":"directorName"
}
Returns a confirmation string.
Delete Account Allow a user to delete their account. DELETE /users/username username None Returns a confirmation string.
