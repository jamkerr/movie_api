YMDB API - Your Movie Database API

With YMDB you can search through movies and save your favourites. This REST API lets you interface with this app as a web service.

Register Allow a new user to register. POST /users None Requires a JSON object with the keys "Username", "Password", "Email", and optionally "Birth_Date". 
                
{
    "Username":"Gregor",
    "Password":"Pass",
    "Email":"gergor@itsme.com",
    "Birth_Date":"1990-01-01"
}
Returns an object. 
                
{
    "Username": "Gregor",
    "Password": "Pass",
    "Email": "gergor@itsme.com",
    "Birth_Date": "1990-01-01T00:00:00.000Z",
    "FavoriteMovies": [],
    "_id": "635eeb0ecad56f9ee61f9899",
    "__v": 0
}
Update User Info Allow a user to update their username, password, email, and birth date. A user can update any combination of these fields while leaving the others unchanged. PUT /users/username username e.g. /users/Gregor Reqires a JSON object with at least one of the keys "Username", "Password", "Email", or "Birth_Date". 
                
{
    "Username":"Sol",
    "Password":"likeasun"
}
Returns an object with updated info. 
                
{
    "_id": "635fb3bb07d71785cea70518",
    "Username": "Sol",
    "Password": "likeasun",
    "Email": "gergor@itsme.com",
    "Birth_Date": "1990-01-01T00:00:00.000Z",
    "FavoriteMovies": [],
    "__v": 0
}
Add Favorite Movie Allow a user to add a movie to their favorites. PUT /users/username/favorites username Requires a JSON object with the key "title". 
                
{
"title":"filmTitle",
"director":"directorName"
}
Returns a confirmation string.
Remove Favorite Movie Allow a user to remove a movie from their favorites. DELETE /users/username/favorites username Requires a JSON object with the key "title". 
                
{
"title":"filmTitle",
"director":"directorName"
}
Returns a confirmation string.
Delete Account Allow a user to delete their account. DELETE /users/username username e.g. /users/Gregor None Returns a confirmation string that the user was deleted.
